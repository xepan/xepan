SHORTCUT DESCRIPTION
F1 This help :)
F2 Toggle Toolbar
F4 Toggle Options
Esc Hide Both Toolbar and Options and de-select any selected component
Ctrl+Shift+Up Select Parent Component, Nothing if already Top Element Selected
Ctrl+Shift+Down Select First Child Component
Ctrl+Shift+Left Select Previous sibling component
Ctrl+Shift+Right Select Next sibling component
Ctrl+Shift+d Delete Selcted component
Ctrl+Shift+< Move selected component before previous sibling
Ctrl+Shift+> Move selcted component after next sibling
Ctrl+s Save :)
Tab Select next element
Shift+Tab Select previous element
Ctrl+Shift+x Cut Selected Component
Ctrl+Shift+c copy Selected Component
Ctrl+Shift+v Paste in last to current selected component
