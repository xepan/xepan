Extended Element component includes various components to give creative look to the site. Currently there are following extended elements are available.

  • Marquee

Marquee

Marquee component allows you to create scrolling container and other xEpan components.

Major features of Marquee :

  • User can add text, images, slideshows and other components in marquee.
  • User can select the direction of marquee like left, right, top, bottom acccording to the need.
  • User can select the behaviour of marquee. Possible values are scroll, slide and alternate. If no value is specified, the default value is scroll.
  • User can manage the speed of marquee like slow, fast, medium.
  • User can change the basic properties like padding, margin, background, border etc. through component tool box.
  • User can live edit the description.
