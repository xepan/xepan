Extended Images

Extended image component is a collection of single images with some enhancement. Currently there is one component available that is image with description.

  • Extended Images

Image with description

Major features of extended images :

  • user can easily select the image.
  • user can apply various css classes.
  • user can manage the text left, right, top, bottom acccording to the need
  • user can change the basic properties like padding, margin, background, border etc. through component tool box.
  • user can live edit the description of image.
