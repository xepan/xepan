Slide show component is a collection of various slide shows. Currently there are following slide shows are available.

  • BootStrap Carausel
  • Awesome Slider
  • Water Wheel Slider
  • Thumbnail Slider

Bootstrap Carousel Slider

The Bootstrap carousel is a flexible, responsive way to add a slider to your site. It is simple, effective and easy to use. It is client side component.


Major features of Bootstrap Carousel slide show :

  • user can easily add images.
  • user can set the height and width of slide show component.
  • user can change the basic properties like padding, margin, background, border etc. through component tool box.


Awesome Slider

The Awesome Slider makes displaying your gallery of images a beautiful experience, by using amazing transition effects ranging from slicing and sliding to fading and folding. There simply is no better way to showcase your amazing work than to make the presentation of it looks just as beautiful. It is server side component.


Major features of Awesome Slider:

  • user can add multiple images.
  • user can add pause time.
  • user can enable or disable mouse hover stop side and side-navbar.
  • user can set the position of paginator.
  • user can easily select the path of the folder of images.
  • user can select the order of images.
  • user can select various effects for the each images.
  • user can change the basic properties like padding, margin, background, border etc. through component tool box.
