Enquiry Form

Enquiry form is a component that includes a page on a Web site that allows users to communicate with the site owner. The page has fields for filling in name, message, address and e-mail. It is a basic form.

Major features of Enquiry Form :

  • User can change the title of the form.
  • User can select the order of the fields.
  • User can mention the e-mail address of the person in receipent, who is receiving the information.
  • User can change the basic properties like padding, margin, background, border etc. through component tool box.
  • User can live edit the description.
  • For more customize form user can use custom form.

Custom Enquiry Form

Custom Enquiry form is a Server Side component that includes a page on a Web site that allows users to communicate with the site owner. The page has fields for filling User Requirment To added Custom Form Fields. It is a More advanced And User Requirment Form.

Major features of Custom Enquiry Form :

  • User can Add Multiple Fields in a single form acoording to our requirement with validation.
  • User can add one or multiple captcha on a single form.
  • User can mention the e-mail address in receipent on Backend Side, who is receiving the information.
  • User can change the basic properties like padding, margin, background, border etc. through component tool box.
  • User can set compulsory Fields Validation Or Not.
  • user can design multiple form.
